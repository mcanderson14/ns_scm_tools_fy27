SCOUT Tools Install Guide

Follow these steps to install the SCOUT staffing helper, roster helper, and calendar refresh workflow.

Use Chrome Or Edge

This guide works best in the browser where Tampermonkey is installed. If a link opens inside ChatGPT/Codex and shows a 431 Request Header Fields Too Large error, click Copy link and paste it into Chrome or Edge.

Open Calendar Refresh Open Staffing Dashboard Install Tampermonkey Open Graph Explorer

Step 1. Install Browser Scripts

Install Tampermonkey in Chrome or Edge, then open each userscript link below and install it when Tampermonkey prompts you.

  1. Install SCOUT staffing helper
  2. Install NetSuite SC roster refresh helper

The roster helper is limited to NetSuite saved search 1311451 and includes an update URL for Tampermonkey.

Step 2. Open Calendar Refresh

Open the shared refresh page and confirm it loads in the browser where Tampermonkey is installed.

Open Calendar Refresh

Step 3. Import The SC Roster

Use the roster saved search to load the current SC list, managers, industries, and emails into the refresh page.

  1. Open the NetSuite SC roster saved search
  2. Click Send roster to refresh on the NetSuite page.
  3. Return to Calendar Refresh and confirm the roster imported.

Step 4. Save A Graph Explorer Token

  1. Open Microsoft Graph Explorer
  2. Sign in with your Oracle Microsoft Office 365 account.
  3. Open API Explorer, then the Access token tab.
  4. Copy the access token.
  5. Paste it into Graph Explorer Access Token on Calendar Refresh.
  6. Click Save token until it expires.

Do not share the token. Treat it like a temporary password.

Step 5. Refresh Calendar Data

  1. In Roster Configuration, click Select all roster or select only the SCs you need.
  2. Keep Days at 21 or less unless you are intentionally testing a larger range.
  3. Click Test first 3 to confirm Graph is working.
  4. Click Refresh selected via token.
  5. Open the dashboard when the refresh completes.

Open Staffing Dashboard

Troubleshooting

  1. If the roster helper appears on the wrong NetSuite page, update the userscript in Tampermonkey and confirm it is version 27.0.0.10 or newer.
  2. If Graph refresh fails with an authentication error, copy a fresh Graph Explorer token.
  3. If the dashboard says no roster is available, repeat Step 3 and import the NetSuite roster again.
  4. If a page looks stale, hard refresh the GitHub Pages tab or add ?v=latest to the URL.
